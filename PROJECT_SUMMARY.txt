PROJECT OVERVIEW
================================================================================

ML Pipeline Studio is a professional, production-grade end-to-end ML Pipeline 
Builder with a visual interface for building, training, and deploying machine 
learning models. This demonstrates advanced ML breadth beyond just LLM APIs, 
showcasing 8+ years of senior engineering expertise.

================================================================================
API ENDPOINTS
================================================================================

Datasets (/api/datasets):
  POST   /upload              Upload new dataset
  GET    /                    List all datasets
  GET    /{id}                Get dataset metadata
  GET    /{id}/stats          Get statistical summary
  GET    /{id}/preview        Preview dataset rows
  DELETE /{id}                Delete dataset

Pipelines (/api/pipelines):
  POST   /run                 Execute ML pipeline
  GET    /                    List pipeline executions
  GET    /{id}                Get pipeline results
  GET    /{id}/status         Get execution status

Models (/api/models):
  GET    /                    List trained models
  GET    /{id}                Get model information
  POST   /{id}/export         Export model (JOBLIB/ONNX)
  GET    /{id}/download/{fmt} Download exported model
  POST   /{id}/predict        Make predictions

System:
  GET    /                    API root info
  GET    /health              Health check
  GET    /api/info            API configuration

GETTING STARTED
================================================================================

1. Navigate to project directory:
   cd /sessions/jolly-great-knuth/mnt/Github-Projects/ml-pipeline-studio

2. View documentation:
   cat README.md

3. Start backend:
   cd backend
   pip install -r requirements.txt

4. Start frontend (new terminal):
   cd frontend
   npm install
   npm run dev

5. Access:
   Frontend: http://localhost:3000
   Backend:  http://localhost:8000
   API Docs: http://localhost:8000/docs
